OudsInlineAlertStatus

sealed class OudsInlineAlertStatus

The status of an OudsInlineAlert. Each status is designed to convey a specific meaning and ensure clarity in communication. It determines the text and the icon colors of the inline alert. It also carries the icon to be displayed in the inline alert. Depending on the status, this icon can be customizable or be a status dedicated icon.
